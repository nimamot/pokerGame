# Amazing Poker Game

Welcome to the Amazing Poker Game!

In this game, two cards are drawn for both the user and the computer. The winner is determined based on the highest card value, second highest card value, or a tie.

## How to play

1. Start the game by running the `main` function.
2. The cards will be generated and displayed for both the user and the computer.
3. The outcome of the game (win, lose, or tie) will be printed.
4. You can choose to play again by typing "y" or exit by typing "n".

## Functions

1. `drawcards`: draws two random cards with values between 2 and 14.
2. `card2str`: converts card values to string type (2 to 10, J, Q, K, A).
3. `printhand`: prints out the card values and player name.
4. `printoutcome`: determines the winner based on the highest card value, second highest card value, or a tie.
5. `main`: calls all the other functions and executes the game.

## Requirements

The code requires the `random` and `time` libraries to run.
